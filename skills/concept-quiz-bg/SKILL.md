# Тест за самопроверка (BG)

Цел: студентът да открие пропуските си. Не трикове — реална проверка на разбиране.

## Вход (питай ако липсва темата)

- Тема/предмет (задължително).
- Ниво — по подразбиране университетски курс във ФМИ.
- Брой въпроси — по подразбиране 5–8.

## Принципи

- Смесвай типове: дефиниция, вярно/невярно, кратък отворен въпрос, приложна задача.
- Подреди от лесно към трудно; покрий различни аспекти, не само един под-сегмент.
- **Отговорите идват накрая, отделно** — за да реши студентът първо сам.
- Към всеки отговор → едно изречение „защо" (обяснение), не само резултата.

## Метод

1. Раздели темата на 4–6 ключови под-идеи.
2. По 1 въпрос за всяка, балансирай типове и трудност.
3. Подготви ключ с кратки обяснения.

## Изход

```
🧠 Тест: [тема]
(Опитай се да отговориш сам, преди ключа долу.)

1. [дефиниция]
2. [Вярно или невярно] [твърдение]
3. [кратък отворен въпрос]
4. [приложна задача]
5. ...

——————————

✅ Ключ с обяснения
1. [отговор] — [едно изречение защо]
2. [Вярно/Невярно] — [обяснение]
3. ...
```

Накрая: „Кажи кои сгреши и ще обясня по-подробно."
