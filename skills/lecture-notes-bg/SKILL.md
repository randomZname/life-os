# Чисти записки от лекция (BG)

Цел: от суров, разбъркан материал → ясни структурирани записки, готови за учене.

## Принципи

- **Не измисляй съдържание.** Работи само с дадения текст. Нечетимо/непълно →
  маркирай с `⚠️ [неясно]`, не гадай.
- Запази терминологията на лектора; добави кратко пояснение при нужда.
- Махни шума (повторения, прекъсвания, „ъъъ"); групирай свързаните идеи.
- Извади дефинициите и формулите отделно — те са най-важни за изпит.

## Метод

1. Прочети целия материал → главна тема + подтеми.
2. Извлечи ключови дефиниции, формули, термини.
3. Подреди подтемите логично (както в учебник).
4. Маркирай местата за доуточняване (`⚠️`).
5. Завърши с 2–4 въпроса за самопроверка.

## Изход

```
📝 [Тема на лекцията]  ·  [предмет/дата ако е дадено]

🔑 Ключови дефиниции
• [термин] — [дефиниция с прости думи]

📌 Основни точки
1. [подтема] — [обобщение в 1–2 изречения]
   – [детайл / пример]
2. ...

🧮 Формули / правила
• [формула] — [какво значи / кога се ползва]

⚠️ За доуточняване
• [неясно място, което да се провери]

❓ За преговор
• [въпрос 1]
• [въпрос 2]
```

Целта е да се учи лесно от записките, не да се преразказва лекцията дума по дума.
