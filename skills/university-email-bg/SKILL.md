# Официален университетски имейл (BG)

Цел: чист, учтив, кратък имейл на български към преподавател или администрация.
Винаги е **чернова** — записва се в `vault/inbox/`, никога не се изпраща автоматично.

## Структура

1. **Тема** — конкретна, 3–7 думи. Пример: „Молба за удължаване на срок — [курс]“.
2. **Обръщение** — „Уважаеми проф. [Фамилия],“ / „Здравейте, [титла] [Фамилия],“.
   При непознат пол/титла: „Уважаеми колеги,“ за администрация.
3. **Контекст (1–2 изречения)** — кой си, кой курс/група, защо пишеш.
4. **Същина** — ясна молба/въпрос. Един имейл = една цел.
5. **Учтиво приключване** — „Благодаря предварително за отделеното време.“
6. **Подпис** — име, фак. номер, специалност, курс.

## Регистър

- Официално „Вие“, без жаргон, без емоджи.
- Кратко > дълго. Без излишни извинения и повторения.
- Конкретни дати във формат ДД.ММ.ГГГГ.

## Шаблон — удължаване на срок

```
Тема: Молба за удължаване на срок — [курс], [задание]

Уважаеми проф. [Фамилия],

Аз съм [Име Фамилия], фак. № [____], специалност [____], [N] курс.
Пиша относно [задание] по [курс] със срок [ДД.ММ.ГГГГ].

Поради [кратка, честна причина] бих искал учтиво да помоля за
удължаване с [N] дни. Готов съм да предам работата до [нова дата].

Благодаря предварително за разбирането и отделеното време.

С уважение,
[Име Фамилия]
Фак. № [____], [специалност], [N] курс
```

## Шаблон — въпрос по материал

```
Тема: Въпрос по [тема] — [курс]

Здравейте, [титла] [Фамилия],

Във връзка с [лекция/упражнение/материал] по [курс] имам въпрос:
[ясно формулиран въпрос, 1–3 изречения].

Благодаря предварително.

С уважение,
[Име Фамилия], фак. № [____]
```

## Финален self-check преди да дадеш черновата

- Една ясна цел? Тема конкретна? Дати верни?
- Учтиво, но без излишно умилкване?
- Подпис с фак. номер?
- Записано в `vault/inbox/`, маркирано като чернова?
