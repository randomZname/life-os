# Учебен план за изпит (BG)

Цел: от предмет + дата на изпит + налично време → конкретен ден-по-ден план.

## Вход (питай ако липсва)

- Предмет и какво влиза (теми/конспект).
- Дата на изпита и колко дни/часове на ден има студентът.
- Текущо ниво по всяка тема (слабо/средно/добро) — ако е известно.
- Има ли минали изпити/задачи за упражнение.

## Метод

1. Раздели конспекта на теми. Оцени тежест × текущо ниво → приоритет.
2. Разпредели темите по дни: трудните и слабите — рано; повторението — накрая.
3. Всеки учебен блок: 25–50 мин учене + 5–10 мин пауза (Pomodoro).
4. Активно учене > препрочитане: решаване на задачи, обяснение на глас, flashcards.
5. Остави последния ден за преговор + решаване на цял минал изпит на време.
6. Заложи буфер — нещо винаги отнема повече.

## Изход

```
📚 [Предмет] — изпит на [ДД.ММ]

Ден 1 ([дата]): [теми] — [N блока] · фокус: [слаба тема]
Ден 2 ([дата]): ...
...
Ден N-1: преговор на всичко + flashcards
Ден N: цял пробен изпит на време + леки теми

Съвет: [1–2 конкретни техники за този предмет]
```

Бъди реалистичен. По-добре по-малко теми наистина научени, отколкото всичко надве-натри.
